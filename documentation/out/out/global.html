This program is an event planner based within an html document. When the program is opened, it will display a calendar with the current month active within it. The user has the ability to navigate the months of the year either by going forward or going backward in chronological order of. The user is able to select any date and the web document will navigate to said url and display a new screen. Within the the new screen it will have two different abilities. The first ability is the capability to add an event for that day. The contents of what makes a date is the host of the events name, the name and description of the event, and finally the times which the event will be held. The times of the events are based on 30 minutes segments with either xx:00 start of xx:30 start. When the event is created it will be displayed within the day that is was created on. The second ability to navigate click on any pre-arranged event and RSVP yourself for the event. When the user RSVPs for the event, they will show up under the attendees column in the event itself.
